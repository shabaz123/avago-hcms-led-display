led-display
rev 1.0 Shabaz October 2016

This code drives Avago HCMS-29xx series displays. It was tested with three such displays connected
in series as per the Avago datasheet.

To use, download the BBBIOlib library from https://github.com/VegetableAvenger/BBBIOlib and
complile it, then type 'make' to build this software.

To display a message, type:
./led-display "Boo!"
